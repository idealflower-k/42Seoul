# minishell

### 프로젝트 설명
- C언어를 사용하여 bash기능 구현

### 맡은 역할(팀원 2명)
- 사용자의 명령어입력을 받아 유효성 검사 및 이진 트리구성
- pipe로직, heredoc구현

### 배운점
- 문자열 데이터 관리
- 이진 트리 구조
- 다중 pipe관리 및 파일 입력 관리
- 약간의 bash에 대한 이해
- 팀원과의 협업
- 수많은 에러 디버깅

### 어려웠던 점
- 너무 다양한 사용자 입력 케이스처리
- "", '' 확장 부분의 에러 개선
- 다중 프로세스환경의 디버깅
